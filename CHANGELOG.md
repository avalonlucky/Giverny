## 2026-06-15 14:07 · v0.10.42（上传校验 / 上传进度 / Toast 队列 / 状态保存反馈）

- 新建任务 / 添加进展的默认日期改为当天（原来写死 2026-05-30），预计交付默认 +3 天，上传文件时间改为实际时间。
- 上传图片预览改用 dataURL 持久化（≤2.5MB），刷新页面后预览不再失效（原 blob URL 刷新即失效）。
- 进展列表 React key 由标题改为 id，标题重复不再渲染错乱。
- 工作台「设计类型工时分布」环形图改为根据实际任务工时动态计算（原数据写死 24.2h）。
- 工时趋势图纵轴上限动态计算（原固定 12h，超出会画出图表外）。
- 「本月总工时」与「计费工时」区分统计；删除假的「较上月 +12%」文案，改为显示导入工时。
- 任务全部删除后不再白屏（空状态保护）。
- localStorage 写入加 try/catch，存储满时不再抛错。

### 新增
- 任务管理页支持删除任务（带确认）。
- 设置页「小时单价」可编辑并持久化，预计收入随单价自动重算。
- 弹窗支持按 ESC 或点击遮罩关闭（统一 ModalShell 组件）。
- 任务列表筛选无结果时显示空状态提示；无任务时「添加进展 / 上传文件」给出引导提示。
- 月报、甲方查看页月份跟随顶栏选择的月份。

- 按钮、卡片、任务行增加过渡动画和悬停反馈；主按钮改渐变并加投影。
- 侧边导航选中态增加品牌色指示条和图标着色。
- 弹窗进出场动画 + 遮罩毛玻璃模糊；Toast 滑入动画。
- 分段筛选选中态白底投影；选中行边框加强区分悬停态。
