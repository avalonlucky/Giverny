# Releases

本文件记录对外可感知的版本要点（精简版）。完整逐项改动见 [CHANGELOG.md](./CHANGELOG.md)。

## v0.11.111 — 2026-06-24

- **工作台任务收束**：已验收任务默认折叠到「已验收 N 个」分区；未完成 / 已验收列表各自先展示一页（15 条）+ 兜底分页，滚动条不再被历史任务拖长。
- **首页科普扩容**：题材新增财经 / 股票、商务礼仪、沟通技巧、爱情 / 约会、治愈系、各国历史、历史上的今天等；篇幅放开（2-5 段、可更长），阅读弹窗正文独立滚动。
- **附件拖拽**：记录进展时可直接把文件拖到附件框上传，拖拽高亮提示。

## v0.11.110 — 2026-06-24

- **改稿轮次显式判定**：记录进展时新增「本次为改稿轮次」开关，只在打开时计入需求人画像，避免把分阶段提交误判为改稿；仅参与画像/AI 分析，不影响计时与结算。
- **静默刷新**：刷新不再每次弹出整页加载卡片，改为先用上次状态快照秒开、后台再无感更新（stale-while-revalidate）。
- **右侧进展对齐**：备注 / 附件与上方进度条右缘对齐；进展正文统一 13.5px。

## v0.11.109 — 2026-06-24

- **本月洞察去卡片化**：去掉补丁底色与描边，与页面融为一体。
- **工时趋势修复 + 按天展示**：修复趋势恒为 0 的 bug（改读分段计时），并由按周改为按天平滑曲线。
- **浮层去描边**：右键菜单 / 月份选择器 / 命令面板 / Toast 去掉发丝线边框，改用柔和投影分离。

## v0.11.106 – v0.11.108 — 2026-06-24（UI 对齐 giverny-redesign，仅换肤）

- 编辑式平面分区调色板：单支鼠尾草绿、纸面三档、墨色发丝线；状态由彩色胶囊改为降饱和小圆点。
- 关键数字（指标 / 日期 / 工时）改用衬线体，年报 / 杂志式质感。
- 侧栏与主区统一同色；浮层先尝试深墨反白，后按需统一改回浅色（深色留待暗黑模式）。

## v0.11.103 – v0.11.105 — 2026-06-24

- 需求人画像新增「延迟率」维度；洞察精简（移除历史视角 / 运行诊断）。
- 结算报表「需求人」字段口径统一为 `requester || contact`。
- 修复验收进展时间轴不显示验收附件缩略图的问题。

## v0.11.92 及更早

- 不计费任务（免费协助）：从创建到交付全程不计入工时与收入，但仍以 ¥0 行出现在结算回单。
- 「已验收」只能通过记录进展走验收闭环达成，不允许改状态直达。
- 分段计时、等待记录、验收进展闭环、PDF 缩略图等基础能力。详见 CHANGELOG。
