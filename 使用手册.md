## 一、这是什么

一个给兼职设计师用的工时管理后台：记录每个月的设计任务、工时、进展和交付文件，月底自动生成银行回执单样式的结算单，一键生成只读链接发给甲方对账。

数据存在 Cloudflare（D1 数据库 + R2 文件存储），换电脑、清浏览器缓存都不会丢。

正式站 `mayeai.com` 只放真实运营数据。预发布测试站及其 D1/R2 资源已删除，功能改版完成本地验证后直接同步正式站。

最新说明：动态时间轴的具体到分钟时间仅管理员可见；普通成员、甲方预览和公开只读报告只显示日期。后台中棕色文字代表“仅管理员可见”的内部信息，对外不可见。

任务导航右侧“进展”面板已按最新原型整理：先看过程记录，再看时间记录、整体进展和交付验收，状态维护与最近操作记录作为低权重后台信息保留。

补录标记是公开解释信息，甲方需要看见；它使用绿色公开提示样式，不归入棕色管理员专属信息。

任务导航右侧“进展记录”现在是完整时间轴：进展更新、任务状态、排期调整和附件动态都在这里，管理员可见的具体时分仍用棕色标记。

整体进展滑块改为自定义绿色进度轨道和白底圆形滑块，拖动时更接近原型效果。

整体进展现在按 10% 档位吸附，并采用“未保存草稿 + 确认”机制：拖动或点档位不会立即写时间轴，只有点击确认才会保存。

管理员可以在任务导航右侧进展时间轴中删除单条任务动态，支持按钮和右键操作；删除仅清理时间轴记录，不回滚任务数据。

删除任务动态时会使用站内确认弹窗，不再出现浏览器原生确认框；确认文案会明确提示删除范围。

项目现在要求：只有影响网站本体的正式更新才同步版本号、GitHub tag 和 Release 更新日志；README、仓库首页、多语言说明、内部协作文档这类不影响网站运行的调整，只做普通 commit / push。涉及明显 UI / 交互变化时，Release 会按需附截图或动图，方便用户理解更新内容。

GitHub README 已补充为多语言项目首页，包含产品概览、任务生命周期和技术架构图，方便新用户或协作者先从仓库首页了解项目。

README 已进一步补全详细说明：新增典型使用场景、模块详解、角色权限、数据口径、文件生命周期和 FAQ，方便用户、甲方和开发接手者理解平台规则。

GitHub README 已改用项目维护者提供的正式站真实截图展示产品界面，不再使用灰条脱敏或示例文案替换图。

## 二、登录规则

打开 mayeai.com 会先看到登录页，有两种登录方式：

### 1. 管理员登录（你自己用）
- **邮箱**：`bh141425@gmail.com`
- **密码**：平台内部管理员密码，可在设置页修改
- 权限：最高权限，比普通用户多一个「口令管理」功能

⚠️ 注意：管理员必须同时输入邮箱和密码，两个都要正确才拥有最高权限。

### 2. 访问口令登录（其他设备 / 协作者用）
- **邮箱留空**，直接输入管理员发放的访问口令（`wk_` 开头）
- 权限：只读浏览为主，不能进行删除等危险操作；管理员才拥有完整管理权限

### 退出登录
左下角「设置」→「退出登录」。登录凭证存在浏览器里，**公共电脑用完务必退出**。

### 登录失效的几种情况
- 口令被管理员停用或删除 → 立即失效，回到登录页
- 口令到期 → 自动失效
- 重新登录即可（管理员账号永远有效）

### 想修改管理员密码怎么办
登录后进入 **设置 → 权限安全 → 账号安全**，填写当前密码和新密码即可。修改后不影响已发放的访问口令。

### 忘记密码怎么办
登录页点击「忘记密码？」后输入管理员邮箱。邮件找回需要先配置发信服务 `RESEND_API_KEY`；未配置时，系统会提示先通过已登录设备在设置页修改密码，或配置发信服务。

### AI 助手密钥
新建任务里的 AI 助手使用 Cloudflare Worker Secret `DEEPSEEK_API_KEY` 调用 DeepSeek。密钥只保存在 Cloudflare 后端，前端页面不会暴露。

如果 AI 助手提示密钥未配置，需要给正式站写入 Secret：
- `npx wrangler secret put DEEPSEEK_API_KEY`

## 三、口令管理（仅管理员可见）

位置：**设置 → 口令管理**

### 申请口令
1. 填备注（如"我的手机"、"iPad"、"协作设计师小王"）
2. 选有效期：**永久 / 7 天 / 30 天 / 90 天**
3. 点「申请口令」→ 自动复制到剪贴板，发给对方即可

### 管理口令
列表里每个口令显示：备注、口令本体（可再次复制）、创建时间、到期时间、最近使用时间、状态。

| 操作 | 效果 |
|------|------|
| 停用 | 该口令立即失效，用它登录的设备下一步操作就会被踢回登录页；可随时「启用」恢复 |
| 删除 | 彻底作废，不可恢复 |

| 状态 | 含义 |
|------|------|
| 有效 | 正常可用 |
| 已停用 | 被手动停用，可恢复 |
| 已过期 | 超过有效期自动失效 |

## 四、日常使用

### 顶栏（所有页面通用）
- **月份切换**：所有页面的数据都跟着所选结算月份走；月份选项根据任务的结算月份自动生成，永远包含当前月
- **结算**：从左侧导航进入结算页，核对月度结算单、锁定结算并生成甲方链接
- **新建任务**：随时可建
- **状态反馈**：操作成功、失败和处理中会用不同的提示样式；侧边栏标题下方的小圆点表示后端连接状态，异常时会变红。

### 工作台
当月总览：总工时、计费工时、预计收入、验收情况四张统计卡；任务明细列表（搜索框 + 筛选下拉，可按计划中/进行中/待验收/已验收过滤）；设计类型工时分布环形图、每周工时趋势图、年度统计柱状图都只统计实际工时，预计开始时间、预计交付时间和预估工时只作为排期参考。

### 任务
- **列表视图 / 日历视图**：进入任务页后，在页面内容区切换，不在左侧导航栏里展开。
- **列表视图**：任务列表会直接展示任务名称、需求摘要、对接人、实际工时、状态、进度、预计开始、预计交付和排期倒计时；搜索框在标题行右侧，状态筛选和「显示作废」在下一行，方便快速扫读。
- **排期提示**：任务行会根据预计开始和预计交付自动显示「距开始」「距交付」「今日 / 明日到期」「已逾期」或「已验收」。列表底部有「临期 / 逾期 / 进行中 / 正常」图例说明颜色含义。这些时间只用于排期提醒，不参与实际工时、收入统计、月报或最终结算。
- **信息 / 进展**：右侧详情区包含两个选项卡。信息页看基础字段；进展页按时间轴记录任务从接受、状态变化、时间段记录、文件上传到验收的全过程。
- **查看任务详情**：详情弹窗用于快速核对任务字段、进度和最近动态；右上角「待验收」状态悬停后会变为「去验收」，点击即可打开验收面板。底部只保留「进展」和「去编辑」两个可操作入口，关闭使用右上角图标。
- **弹窗关闭规则**：所有弹窗默认不会因为点击外部空白处或按 Esc 关闭，避免误点丢内容；需要关闭时点击右上角关闭图标或弹窗里的取消按钮。新建任务、记录进展和确认验收会保留未保存草稿，保存 / 确认后才清空。
- **右键快捷操作**：在工作台任务行或任务页列表中右键任务，可快速查看详情、记录进展、去验收、调整进度或复制已锁定月份的甲方分享链接；「状态」会以二级菜单收纳进行中、待验收、挂起和终止。挂起 / 终止会要求填写原因。
- **空白处右键新建**：在工作台任务列表或任务页列表空白处右键，可以直接打开「新建任务」菜单；未登录管理员时会先提示登录。
- **AI 优化进展 / 验收备注**：在「记录进展」弹窗的新增进展标题右侧、以及「确认验收」弹窗的验收备注标题右侧，都有 AI 图标。点击后，系统会结合当前填写内容、任务需求、任务相关文件、过程附件 / 验收附件和最近进展生成建议；建议需要手动点「采用建议」才会写回输入框。
- **删除任务动态**：在「记录进展」弹窗的进展时间轴中，管理员可右键单条动态，或悬停后点击删除图标，删除前会出现站内确认弹窗；删除只移除时间轴记录，不回滚任务字段、文件或工时数据。
- **任务不允许删除**：任务会影响工时、收入和税务统计，正式运营后不做删除。暂时不做的任务选择「挂起」，确定不做的任务选择「终止」并填写原因。
- **作废任务**：测试任务或误录任务可通过右键菜单作废，系统会弹出站内确认框，可填写作废原因；作废后不计入工时、收入和结算，也不出现在工作台、日历和月报里，但保留历史记录和关联文件。
- **查看 / 删除作废任务**：任务页默认隐藏作废任务，可打开「显示作废」查看；只有已作废任务才允许永久删除，删除前仍会二次确认。
- **排序规则**：工作台和任务列表先按“结算月份”筛选，再按“预计开始时间 / 开始日期倒序”排列；同一天内按创建时间倒序，新建的排在更上面。预计开始时间只影响列表排序和排期提醒，不参与统计、工时计算或结算。
- **计划时间**：新建任务时可填写预计开始时间到分钟，例如 `2026/06/13 10:10`。新建面板支持双向推算：打开「预计开始时间」旁的开关时，系统按开始时间 + 预估工时推算交付时间；打开「预计交付时间」旁的开关时，系统按交付时间 - 预估工时倒推开始时间。未打开开关的一侧只读展示。预计开始时间、预计交付时间和预估工时只用于排期参考，不影响本月洞察、实际工时、收入统计、月报或最终结算。
- **AI 优化需求**：新建任务时，「任务具体需求」右上角有 AI 图标。填写项目名称或原始需求后点击，可让 DeepSeek 助手把口语化描述整理成「设计背景 / 设计要求 / 输出文件」三段任务单语言，并推荐设计类型。建议只会展示给管理员，采用文案、采用分类或新增分类都需要手动确认。
- **补录历史任务**：新建任务弹窗右上角有「补录」文字和一个极简小开关，默认关闭。只有特殊补录场景才打开，打开后在开关旁显示「计入月份」下拉，不会占用主表单空间。例如任务实际 5 月完成，但需要放入 6 月工资一起结算，就把预计开始时间按真实 5 月填写，同时把计入月份选为 6 月。这样它会出现在 6 月工作台、任务列表和 6 月月度结算单中，并在日期旁标记“补录”。
- **结算月份**：普通新任务默认计入顶部当前月份；修改预计开始时间或预计交付时间不会自动改结算月份，需要调整归属时请显式修改「结算月份」。
- **补录状态建议**：新建任务不选择阶段，也不直接操作验收。已经完成、只需要补结算的任务，创建时填写必要的「补录说明」，创建后进入任务详情点击「确认验收」，再补实际工时、验收备注和验收文件。还没开始或正在做的任务，创建后按真实进度在任务详情或任务列表中调整状态。

### 工作台右侧任务明细卡
在工作台的任务明细里**点任意一行**，右侧明细卡同步显示该任务：
- 任务状态（徽章）、**预计交付时间**（临期/逾期会标黄/标红）、实际工时、对接人、验收人
- 整体进展条
- **上传附件**入口：选完文件自动上传并关联到这个任务
- **动态时间轴**：最近 4 条操作记录
- 底部**「查看更多」**按钮 → 展开完整详情弹窗

### 任务详情弹窗（点「查看更多」展开）
- 展示任务基础字段、预计开始、预计交付、任务状态、当前进度、实际工时、结算月份和最近进展。
- 不再提供单独文件上传入口。过程附件从「进展」里上传；验收附件在「去验收」终审面板里补传。
- 待验收任务右上角状态可直接进入验收，确认前仍会核对工时、备注和验收附件。
- **最近动态**：展示最近操作记录；完整过程记录请进入「进展」查看。

### 交付日期提醒
- 新建/编辑任务时填「预计交付时间」，系统自动盯着：
  - **临期**（3 天内要交付）：黄色标记
  - **已逾期**（过了交付日还没验收）：红色标记
- 工作台顶部会出现黄色提醒条（统计所有月份的临期/逾期任务），点击直接跳到任务页
- 已验收和不计费的任务不参与提醒

### 任务状态规则
| 状态 | 含义 | 计费 |
|------|------|------|
| 计划中 | 已立项还没开工（新建任务默认） | ❌ 不进结算单 |
| 进行中 | 正在做 | ✅ |
| 挂起 | 暂时暂停，必须填写挂起原因 | ❌ 暂不计入结算 |
| 待验收 | 做完等甲方确认 | ✅ |
| 已验收 | 甲方确认完成 | ✅ |
| 终止 | 需求取消或确定不继续，必须填写终止原因 | ❌ 不进结算单 |
| 不计费 | 帮忙性质，不收钱 | ❌ |

### 进展
- 进展已合并到任务详情里，不再作为左侧独立导航。
- 可以记录某个时间段做了什么，例如 `14:00-15:30 修改主视觉文案层级`。这些时间段会纳入任务实际工时。
- 状态变化、接受任务、上传文件、确认验收等操作也会写入时间轴。
- 确认验收时，如果已经修改验收备注、调整时间段或上传验收文件但还没点「确认验收」，关闭弹窗前会先提示确认；附件上传中或系统计算工时为 0 时不能提交验收。

### 文件库
- 文件库不是单独上传入口，而是自动汇总任务生命周期中的文件。
- 文件来源包括：任务过程文件、进展附件、验收时上传的最终稿和验收文件。
- 文件按“项目 / 任务”归档，点击某个任务后查看该任务下所有关联文件。
- 支持搜索文件名、任务名和标签；支持按月份筛选。
- 点击文件卡片会打开右侧信息栏，可重命名、添加多个标签、预览、打开或删除。
- 右键点击文件卡片可快速预览、打开、重命名、添加标签、下载源文件或删除；重命名和添加标签会自动聚焦右侧信息栏。
- 删除文件只用于处理误传文件。点击删除后会弹出站内二次确认框，确认后才会删除 D1 文件记录、R2 源文件和预览图。
- 已验收、已发给甲方或可能用于回溯的文件建议保留。
- 空格键可以快速预览当前选中文件。
- PSD、AI、PDF、Word、Excel、PPT、MP4 和图片尽量提供在线预览；部分源文件格式依赖浏览器和预览图能力，必要时仍可打开或下载源文件。

### 结算（月报核心流程）
月底结算的标准流程：

1. 切到要结算的月份，核对回执单上的任务和工时
2. 点 **「锁定结算并生成甲方链接」** → 金额快照存档，链接自动复制
3. 把链接（`mayeai.com/share/xxxx`）发给陈义君
4. 需要 PDF 的话点「导出 PDF」→ 打印窗口选「另存为 PDF」（文件名自动带月份）

**锁定规则：**
- 每个月只有一条结算记录；锁定后又改了数据，再点一次锁定即可更新金额，**链接不变**（已发出的链接不会失效）
- 锁定的金额是快照——以后改时薪不影响已锁定的月份
- 「结算历史」列出所有锁定过的月份，随时复制链接或打开查看
- **甲方查看回执**：甲方打开过链接后，结算历史会显示「甲方已查看 N 次（最近时间）」；没打开过显示「甲方尚未查看」——月底催验收前先看一眼

### 年度统计（工作台底部）
- 全年 12 个月的计费工时柱状图，🔒 表示该月已锁定结算
- 累计工时、累计收入一目了然（已锁定月份按结算快照计，未锁定月份按当前时薪实时算）
- 鼠标悬停柱子可看每月明细

## 五、计费规则速查

- 预计收入 = 计费工时 × 小时单价
- 计费工时 = 当月所有任务实际工时之和，**排除**「不计费」和「计划中」状态
- 统计只读取 D1 中的正式任务数据和已锁定月报快照，不再包含任何硬编码导入工时
- 结算单序号、人民币大写金额、电子印章均自动生成

## 六、运营与数据安全

### 正式站
- 正式站：`https://mayeai.com`，只录入真实任务、真实工时和真实交付文件。
- 预发布测试站已下线，测试 D1/R2 资源也已删除。
- 每次功能改动完成本地验证后直接同步正式站。

### 文件库清理原则
- 当前没有自动按月删除机制，避免误删当月文件或影响甲方月报。
- 推荐每月结算后执行一次人工复核：确认月报已锁定、甲方已验收、备份已导出，再考虑清理低价值过程文件。
- 最保险的做法不是直接删除，而是后续增加“归档 / 冷存储 / 待删除队列”机制，给文件至少 30 天反悔期。
- 最终稿、验收文件、已发给甲方的文件不建议删除；源文件是否删除应以“是否仍可能返修 / 补交”为准。
